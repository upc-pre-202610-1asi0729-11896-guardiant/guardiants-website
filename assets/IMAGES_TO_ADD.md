# Images to add

Add your images in this folder using these exact filenames (or update the paths in `index.html`).

## Required filenames

- `hero-bg.jpg` (used as the Hero background via CSS)
- `mission.jpg`
- `vision.jpg`
- `how-natural-person.jpg`
- `how-companies.jpg`
- `telemetry-dashboard.jpg`
- `contact.jpg`

## Notes

- The page uses grayscale styling (`filter: grayscale(100%)`). If you want full color, remove the CSS `filter` rules for `.media img`, `.card-image img`, and `.contact-card img` in `index.html`.
